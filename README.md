# outfit

This repository is configured for development and code submission through GitHub:

- GitHub repository: <https://github.com/cumtfc/outfit>
- Default working branch in this environment: `work`

## Basic workflow

```bash
git status
git add <changed-files>
git commit -m "Describe the change"
git push origin work
```

Before pushing, confirm that the `origin` remote points to `https://github.com/cumtfc/outfit.git`.
